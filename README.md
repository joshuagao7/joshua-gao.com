

Joshua's Personal Website

IDEAS: create a selection/radar chart, plotting out mapping of time: selecting money, health, time with family, etc. and interact with this

A modern personal website built with Next.js, React, and TypeScript.

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **CSS Modules** - Styled components

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Modern, responsive design
- Smooth animations and transitions
- SEO-friendly structure
- Fast performance with Next.js optimization

## Customization

Edit the content in `app/page.tsx` to personalize your website. Update styles in `app/globals.css` to match your brand.

## Deployment

This site can be easily deployed to:
- **Vercel** (recommended for Next.js)
- **Netlify**
- **GitHub Pages** (with static export)
